# 🛒 E-commerce Web Application

## 🚀 Overview

This is a full-stack E-commerce application currently under development.

The project includes:

* A **React frontend** for user interaction
* A **Django backend** for API and data handling (in progress)

---

## 🧰 Tech Stack

### Frontend

* React
* JavaScript
* CSS

### Backend

* Django
* Django REST Framework (planned)

---

## ✨ Current Features (Frontend)

* 🏠 Home page
* 🛍️ Product listing (Shop Page)
* 🛒 Add to Cart functionality
* ❤️ Wishlist page
* 💳 Purchase page UI

---

## ⚙️ Backend Status

* Backend structure is created using Django
* API endpoints are under development
* Frontend and backend integration is **not yet completed**

---

## 📁 Project Structure

```
Ecommerce/
   frontend/   → React application
   backend/    → Django backend
```

---

## 🧪 How to Run the Project

### ▶️ Frontend

```
cd frontend
npm install
npm start
```

### ▶️ Backend

```
cd backend
python manage.py runserver
```
